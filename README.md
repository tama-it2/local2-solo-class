##東京都　大田区##
*大森と鎌田の文字を取った名称
*埋立地
*シンゴジラの侵入地点　第一形態は鎌田君の愛称がある
*豚に改造された信楽焼の狸が置かれたとんかつ屋がある
*ドラマなどで頻繁に事件現場となる
